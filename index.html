Build your web app in a few minutes.

The main goal is to improve and accelerate the development process. Built on top of jQuery, this toolbox brings a couple of known visual patterns for the web and takes advantages of the new HTML5 and CSS3 features.

Chico-UI team is available for any feedback or suggestions. We want to share this tool with all designers and developers to make the web beautiful.

Give it a try!

  • We love the open web.

    We want to collaborate to make the web more beautiful, with HTML5, CSS3, JavaScript.

  • Open source

    We believe in the open source, you can play with Chico's code is in github.

  • Crossbrowsing

    Still supporting old browsers: IE6+, FF2+, Chrome, Safari, Opera.
