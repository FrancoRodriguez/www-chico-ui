Classdetail

Parameters

  • conf: Object, Optional. Object with configuration properties.
  • conf.width: Number || String, Optional. Sets width property of the component's layout. By default, the width is elastic.
  • conf.pagination: Boolean, Optional. Shows a pagination. By default, the value is false.
  • conf.arrows: Boolean, Optional. Shows arrows icons over or outside the mask. By default, the value is "outside".
  • conf.asyncData: Array, Optional. Defines the content of each item that will be load asnchronously as array.
  • conf.asyncRender: Function, Optional. The function that receives asyncData content and must return a string with result of manipulate that content.
  • conf.fx: Boolean, Optional. Enable or disable UI effects. By default, the effects are enabled.
  • conf.maxItems: Number, Optional. (Since 0.10.6) Set the max amount of items to show in each page.
  • conf.page: Number, Optional. (Since 0.10.6) Initialize the Carousel in a specified page.

Returns

  • itself

Examples

var foo = $("#example").carousel();
var foo = $("#example").carousel({
    "width": 500,
    "pagination": true,
    "arrows": "over"
});
var foo = $("#example").carousel({
    "asyncData": [
        {"src": "a.png", "alt": "A"},
        {"src": "b.png", "alt": "B"},
        {"src": "c.png", "alt": "C"}
    ],
    "asyncRender": function (data) {
        return "";
    }
});

Methods detail

addArrows (private)

Add arrows to DOM, bind these event and change the flag "arrowsCreated".

addItems (private)

Move items from queue to collection.

Parameters
  • amount: Number. Amount of items that will be added.

addPagination (private)

Create the pagination on DOM and change the flag "paginationCreated".

arrows (public)

Allow to manage or disable the "Next" and "Previous" buttons flow ("over" the mask, "outside" it or "none"). (Since 0.10.6).

Examples
foo.arrows("outside");
// or
foo.arrows(true);
foo.arrows("over");
foo.arrows("none");
or
foo.arrows(false);
Parameters
  • config (Since : String || Boolean. CSS transition properties. By default it's "outside".
Returns
  • Chico UI Object

arrowsFlow (private)

Allows to render the arrows over the mask or not.

Parameters
  • config: String || Boolean. Defines the arrows behavior. It can be "outside", "over", "none", true or false. By default it's "outside".

createLayout (private)

Makes ready the component structure.

draw (private)

Trigger all recalculations to get the functionality measures.

getPage (private)

Calculates the page corresponding to a specific item.

goToPage (private)

Updates all necessary data to move to a specified page.

Parameters

loadAsyncItems (private)

Analizes if next page needs to load items from queue and execute addItems() method.

Defines the sizing behavior of Carousel. It can be elastic and responsive or fixed.

switchPagination (private)

Updates the selected page on pagination.

translate (private)

Moves the list corresponding to specified displacement.

updateARIA (private)

Set WAI-ARIA properties to each item depending on the page in which these are.

updateArrows (private)

Check for arrows behavior on first, last and middle pages, and update class name and ARIA values.

updatePagination (private)

updateResponsiveness (private)

Calculates and set the size of items and its margin to get an adaptive Carousel.
