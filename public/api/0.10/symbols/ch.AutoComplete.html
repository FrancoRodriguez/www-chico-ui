Class detail

Parameters

  • conf: Object, Optional. Object with configuration properties.
  • conf.open: Boolean, Optional. Shows the autoComplete open when component was loaded. By default, the value is false.
  • conf.fx: Boolean, Optional. Enable or disable UI effects. By default, the effects are disable.

Returns

  • itself

Examples

// Create a new autoComplete with configuration.
var me = $(".example").autoComplete({
    "open": true,
    "fx": true
});
// Create a new autoComplete without configuration.
var me = $(".example").autoComplete();
