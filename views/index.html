Build your web app in a few minutes.

The main goal is to improve and accelerate the development process. Built on top of jQuery, this open source toolbox brings a couple of known visual patterns for the web and takes advantages of the new HTML5 and CSS3 features. We want to share this tool with all designers and developers to help making the web more beautiful.

You can get in touch with us through Email, Google Groups, Twitter or Faceboook. Also you can see our coding in Github, or grab the project in either zip or tar formats.
